גיליון Google Sheets — חובה

📁 יש לכם Excel מקומי? פתחו sheets.google.com ← + ← Import ← העלו את קובץ ה-Excel שלכם. Google תמיר אותו אוטומטית.

הגיליון צריך לכלול עמודות: שם תלמיד, שם משלם, מספר טלפון, סכום. המערכת מזהה את העמודות אוטומטית.

💡 שם משלם = מי שמשלם בפועל (הורה / תלמיד). אם התלמיד משלם בעצמו — כתבו את שמו פעמיים.
💡 מספר טלפון — הזינו עם אפס בהתחלה (0524...). הגדירו את העמודה כ-טקסט (Format → Number → Plain text) כדי שהאפס יישמר.

מספר הטלפון שלכם (עם אפס: 050...) — לצורך קבלת התראות מנוי ואישורי חידוש מהמערכת.

אייפון: Shortcuts מובנית. תקבלו הנחיות קצרות.

⚙️ הקמה — 3 צעדים

1
התקנת Add-on
פתחו את הגיליון שלכם ← Extensions ← Add-ons ← Get add-ons ← חפשו "LessonDone Lite" ← Install
אשרו את ההרשאות. ההרשאה מאפשרת ל-Add-on לעדכן את הגיליון שלכם בלבד. אף אחד אחר לא מקבל גישה.
רענון הדף ← תפריט חדש מופיע: 💰 LessonDone Lite ← לחצו הפעלה ראשונה
המערכת תזהה את העמודות שלכם אוטומטית ותוסיף עמודות תשלום בסוף.

🔒 למה זה בטוח?

ה-Add-on רץ בחשבון Google שלכם. הנתונים לא עוזבים את הגיליון שלכם. אף אחד — כולל מפתח המערכת — לא יכול לראות את המידע שלכם.

2
חיבור הטלפון
📱 אנדרואיד — MacroDroid
1. התקינו MacroDroid מ-Google Play (חינם)
2. תקבלו קובץ .macro מוכן ב-WhatsApp
3. לחצו על הקובץ ← ייובא אוטומטית
4. סיום!
🍎 אייפון — Shortcuts
1. תקבלו לינק ב-WhatsApp ← לחצו ← "Add Shortcut"
2. הזינו את הקוד שקיבלתם (פעם אחת)
3. Shortcuts ← Automation ← + ← Message ← Contains ← ש"ח
4. Run Shortcut ← LessonDone Lite ← Done
💡 MacroDroid / Shortcuts חייבים לרוץ ברקע. ודאו שהטלפון לא מכבה אותם.
3
בדיקה
בקשו ממישהו לשלוח לכם 1₪ ב-Bit.

💬 תזכורות WhatsApp (לא חובה)

LessonDone Lite עובד מצוין גם בלי WhatsApp. אם תרצו — אפשר להוסיף תזכורות תשלום אוטומטיות.

1
יצירת חשבון GREEN API
פתחו green-api.com ← לחצו Register

הזינו אימייל וסיסמה ← אשרו את המייל שנשלח אליכם
💡 תקופת ניסיון חינמית — לא צריך כרטיס אשראי להתחלה.
📋 תוכניות GREEN API:
• Developer (חינמי) — שליחת תזכורות לעד 3 מספרי טלפון שונים בחודש בלבד. מתאים אם יש לכם מעט הורים.
• Start (~$12/חודש) — ללא הגבלת מספרים. מתאים למורה עם תלמידים רבים.
ניתן להתחיל בחינמי ולשדרג בכל עת.
2
יצירת Instance וסריקת QR
לאחר הכניסה ← לחצו Create Instance ← בחרו Developer (חינם)

← מופיעים: idInstance ו-apiToken — שמרו אותם!
← לחצו Scan QR
← פתחו WhatsApp בטלפון שלכם
← שלוש נקודות ← Linked Devices ← Link a Device
← סרקו את הקוד שעל המסך

✅ הנקודה הופכת ירוקה = מחוברים!
⚠️ הטלפון חייב להיות מחובר לאינטרנט כדי שהתזכורות יישלחו. לא צריך שהמסך יהיה דלוק.
3
הזנת פרטים בגיליון
בגיליון הניהול (גיליון מורות) ← מצאו את השורה שלכם

תפריט 💰 LessonDone Lite ← 💬 הגדר WhatsApp

הזינו:
• idInstance — המספר שקיבלתם (למשל: 1234567890)
• apiToken — הקוד הארוך שקיבלתם
💡 ודאו שעמודת מספר טלפון בגיליון מלאה — לשם נשלחות התזכורות.
4
בדיקת חיבור
תפריט 💰 LessonDone Lite ← 💬 בדוק חיבור WhatsApp

✅ מחובר = הכל מוכן, תזכורות ישלחו אוטומטית בכל יום 18:00
❌ לא מחובר = חזרו לשלב 2 וסרקו מחדש

🔔 מתי נשלחות תזכורות?

• כל יום בשעה 18:00 — לא בשישי ושבת
• רק להורים עם חוב פתוח מעל 3 ימים
• מקסימום 3 תזכורות per תלמיד
• ההודעה כוללת את דרכי התשלום שהגדרתם
